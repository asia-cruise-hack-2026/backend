| 파일 | 설명 |

| `tamnao_products.json` | 전체 270개 구조화 데이터(지도/앱 연동용) |
| `tamnao_products.csv` | 엑셀용(UTF-8 BOM). 좌표·거리·최근접항·주소 포함 |
| `탐나오_관광지_주소정리.md` | 본 보고서 |

**JSON 레코드 스키마**
```
prdtNum, prdtDiv(COUP/TOUR), category(관광지·레저/맛집),
name, poi, lat, lng, addr_short, addr_full,
dist_jeju_km, dist_gangjeong_km, nearest_port,
url(탐나오 상세페이지)
```
`url`이 곧 지도 핀 ↔ 상품 상세페이지를 잇는 연결고리다.

## 6. 다음 단계(지도 연동)
1. 이 JSON을 그대로 카카오맵(웹사이트가 이미 사용) 마커 데이터로 사용 → 지오코딩 불필요.
2. 기항지 선택(제주항/강정항) + 사용자 취향 필터 → `dist_*_km`와 `category`로 후보 필터링.
3. 마커 클릭 → 카드 하이라이트 → `url`로 탐나오 상세 연결(양방향 연동).
4. 정밀 이동시간이 필요하면 직선거리 대신 카카오 길찾기 API로 교체.
